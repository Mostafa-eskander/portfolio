# 🌟 Portfolio Website

Personal portfolio website showcasing projects, skills, and a contact form. Designed to be fully responsive and modern, perfect for presenting your work to potential clients or employers.

---

## ✨ Features

- ✅ **Responsive Design:** Looks great on desktop, tablet, and mobile.  
- ✅ **Projects Showcase:** Display your past work with images and descriptions.  
- ✅ **Skills Section:** Highlight your technical skills and technologies.  
- ✅ **Contact Form:** Users can send messages directly from the website.  
- ✅ **Smooth Navigation:** Navbar toggle for mobile menu

## 🚀 Getting Started

Follow these steps to run the project locally:

1️⃣ Clone the repository
```
git clone https://github.com/Mostafa-eskander/protgolio.git
```
3️⃣ Install dependencies
```
npm install
```
4️⃣ Run the development server
```
npm run dev
```
The app will run on:
```
http://localhost:5173
```

---

### 🌐 Deployment

This project is deployed using Vercel.
To deploy your own version:
```
npm run build
```
Then upload the dist folder to Vercel.
